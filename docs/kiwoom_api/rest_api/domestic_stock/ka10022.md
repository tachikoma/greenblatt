# [ka10022] 잔량율급증요청

## 개요
- **Category**: rest_api/domestic_stock
- **Method**: POST
- **URL**: /api/dostk/rkinfo

## 요청 헤더
| 필드 | 한글명 | Type | 필수 | 설명 |
| :--- | :--- | :--- | :--- | :--- |
| api-id | TR명 | String | Y |  |
| authorization | 접근토큰 | String | Y | 토큰 지정시 토큰타입("Bearer") 붙혀서 호출   예) Bearer Egicyx... |
| cont-yn | 연속조회여부 | String | N | 응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅 |
| next-key | 연속조회키 | String | N | 응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅 |

## 요청 바디
| Element | 한글명 | Type | 필수 | 설명 |
| :--- | :--- | :--- | :--- | :--- |
| `mrkt_tp` | 시장구분 | String | Y | 001:코스피, 101:코스닥 |
| `rt_tp` | 비율구분 | String | Y | 1:매수/매도비율, 2:매도/매수비율 |
| `tm_tp` | 시간구분 | String | Y | 분 입력 |
| `trde_qty_tp` | 거래량구분 | String | Y | 5:5천주이상, 10:만주이상, 50:5만주이상, 100:10만주이상 |
| `stk_cnd` | 종목조건 | String | Y | 0:전체조회, 1:관리종목제외, 5:증100제외, 6:증100만보기, 7:증40만보기, 8:증30만보기, 9:증20만보기 |
| `stex_tp` | 거래소구분 | String | Y | 1:KRX, 2:NXT 3.통합 |
