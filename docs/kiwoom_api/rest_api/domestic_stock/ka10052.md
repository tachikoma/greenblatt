## 개요
- **Category**: rest_api/domestic_stock
- **Method**: POST
- **URL**: /api/dostk/stkinfo

| 필드 | 한글명 | Type | 필수 | 설명 |

| api-id | TR명 | String | Y |  |
| authorization | 접근토큰 | String | Y | 토큰 지정시 토큰타입("Bearer") 붙혀서 호출   예) Bearer Egicyx... |
| cont-yn | 연속조회여부 | String | N | 응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅 |

## 요청 바디
| Element | 한글명 | Type | 필수 | 설명 |
| :--- | :--- | :--- | :--- | :--- |
| `mmcm_cd` | 회원사코드 | String | Y | 회원사 코드는 ka10102 조회 |
| `stk_cd` | 종목코드 | String | N | 거래소별 종목코드 (KRX:039490,NXT:039490_NX,SOR:039490_AL) |
| `mrkt_tp` | 시장구분 | String | Y | 0:전체, 1:코스피, 2:코스닥, 3:종목 |
| `qty_tp` | 수량구분 | String | Y | 0:전체, 1:1000주, 2:2000주, 3:, 5:, 10:10000주, 30: 30000주, 50: 50000주, 100: 100000주 |
| `pric_tp` | 가격구분 | String | Y | 0:전체, 1:1천원 미만, 8:1천원 이상, 2:1천원 ~ 2천원, 3:2천원 ~ 5천원, 4:5천원 ~ 1만원, 5:1만원 이상 |
| `stex_tp` | 거래소구분 | String | Y | 1:KRX, 2:NXT 3.통합 |
