| Element | 한글명 | Type | 필수 | 설명 |

| `mrkt_tp` | 시장구분 | String | Y | 코스피- 거래소구분값 1일경우:P00101, 2일경우:P001_NX01, 3일경우:P001_AL01 코스닥- 거래소구분값 1일경우:P10102, 2일경우:P101_NX02, 3일경우:P101_AL02 |
