# [ka10001] 주식기본정보요청

## 1. 개요
- **Category**: rest_api/domestic_stock/
- **Method**: POST
- **URL**: /api/dostk/v1/stk/info
- **Content-Type**: application/json;charset=UTF-8
- **설명**: 국내주식의 기본적인 종목 정보 및 펀더멘털 데이터를 조회합니다. PER, ROE 등 펀더멘털 데이터는 외부 벤더사 제공 데이터로 실적 발표 시즌 등에 업데이트됩니다.

## 2. Request

### Header
| Element | 한글명 | Type | Size | Required | Description |
| :--- | :--- | :--- | :--- | :--- | :--- |
| api-id | TR명 | String | 10 | Y | ka10001 |
| authorization | 접근토큰 | String | 1000 | Y | Bearer {token} 형식 |
| cont-yn | 연속조회여부 | String | 1 | N | 응답의 cont-yn이 Y일 경우 다음 요청 시 세팅 |
| next-key | 연속조회키 | String | 50 | N | 응답의 next-key 값을 다음 요청 시 세팅 |

### Body
| Element | 한글명 | Type | Size | Required | Description |
| :--- | :--- | :--- | :--- | :--- | :--- |
| stk_cd | 종목코드 | String | 20 | Y | 거래소별 종목코드 (예: KRX:005930) |
