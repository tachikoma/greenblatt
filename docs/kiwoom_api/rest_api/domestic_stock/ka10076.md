| Element | 한글명 | Type | 필수 | 설명 |
| :--- | :--- | :--- | :--- | :--- |
| `stk_cd` | 종목코드 | String | N |  |
| `qry_tp` | 조회구분 | String | Y | 0:전체, 1:종목 |
| `sell_tp` | 매도수구분 | String | Y | 0:전체, 1:매도, 2:매수 |
| `ord_no` | 주문번호 | String | N | 검색 기준 값으로 입력한 주문번호 보다 과거에 체결된 내역이 조회됩니다. |
