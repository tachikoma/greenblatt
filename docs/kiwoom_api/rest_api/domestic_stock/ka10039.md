| Element | 한글명 | Type | 필수 | 설명 |
| :--- | :--- | :--- | :--- | :--- |

| `trde_qty_tp` | 거래량구분 | String | Y | 0:전체, 5:5000주, 10:1만주, 50:5만주, 100:10만주, 500:50만주, 1000: 100만주 |
