# [ka20004] 업종틱차트조회요청

## 개요
- **Category**: rest_api/industry
- **Method**: POST
- **URL**: /api/dostk/chart

## 요청 헤더
| 필드 | 한글명 | Type | 필수 | 설명 |
| :--- | :--- | :--- | :--- | :--- |
| api-id | TR명 | String | Y |  |
| authorization | 접근토큰 | String | Y | 토큰 지정시 토큰타입("Bearer") 붙혀서 호출   예) Bearer Egicyx... |
| cont-yn | 연속조회여부 | String | N | 응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅 |
| next-key | 연속조회키 | String | N | 응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅 |
| api-id | TR명 | String | Y |  |
| cont-yn | 연속조회여부 | String | N | 다음 데이터가 있을시 Y값 전달 |
| next-key | 연속조회키 | String | N | 다음 데이터가 있을시 다음 키값 전달 |

## 요청 바디
| Element | 한글명 | Type | 필수 | 설명 |
| :--- | :--- | :--- | :--- | :--- |
| `inds_cd` | 업종코드 | String | Y | 001:종합(KOSPI), 002:대형주, 003:중형주, 004:소형주 101:종합(KOSDAQ), 201:KOSPI200, 302:KOSTAR, 701: KRX100 나머지 ※ 업종코드 참고 |
| `tic_scope` | 틱범위 | String | Y | 1:1틱, 3:3틱, 5:5틱, 10:10틱, 30:30틱 |

## 응답 바디
| Element | 한글명 | Type | 설명 |
| :--- | :--- | :--- | :--- |
| `inds_cd` | 업종코드 | String |  |
| `inds_tic_chart_qry` | 업종틱차트조회 | LIST |  |
| `- cur_prc` | 현재가 | String | 지수 값은 소수점 제거 후 100배 값으로 반환 |
| `- trde_qty` | 거래량 | String |  |
| `- cntr_tm` | 체결시간 | String |  |
| `- open_pric` | 시가 | String | 지수 값은 소수점 제거 후 100배 값으로 반환 |
| `- high_pric` | 고가 | String | 지수 값은 소수점 제거 후 100배 값으로 반환 |
| `- low_pric` | 저가 | String | 지수 값은 소수점 제거 후 100배 값으로 반환 |
| `- pred_pre` | 전일대비 | String | 현재가 - 전일종가 |
| `- pred_pre_sig` | 전일대비 기호 | String | 1: 상한가, 2:상승, 3:보합, 4:하한가, 5:하락 |
