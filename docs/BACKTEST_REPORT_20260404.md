## 1. 개요

본 보고서는 `feature/calc-fix` 브랜치의 수정된 랭킹 로직을 적용한 백테스트 결과를 기록합니다.  
`develop` 브랜치 대비 두 가지 핵심 수정 사항이 포함되며, 모멘텀 가중치(`momentum_weight`) 파라미터 민감도 분석 결과도 함께 수록합니다.

## 2. 코드 변경 사항 (`feature/calc-fix`)

### 커밋 요약

| 커밋 | 내용 |
|------|------|
| `c38e9cf` | fix: 그린블라트 순위 로직을 등수 합산 방식으로 통일 |
| `7a25b26` | feat: 2단계 혼합 순위 방식 적용 (그린블라트 + momentum_weight 오버레이) |

### 변경 내용

#### 변경 1: ROE 가중치 제거

```python
# BEFORE (develop)
df['total_rank'] = df['rank_per'] + df['rank_pbr'] + (df['rank_roe'] * 1.5)

# AFTER (calc-fix)
df['total_rank'] = df['rank_per'] + df['rank_pbr'] + df['rank_roe']
```

ROE에 1.5배 부여하던 임의 가중을 제거, **순수 그린블라트 원형(등수 단순 합산)**으로 복귀.

#### 변경 2: 2단계 혼합 방식 도입

```python
# BEFORE (develop) — 프로파일별 정규화 백분위 가중합
large_cap = 0.20 * value_score + 0.60 * rank_roe_norm + 0.20 * rank_mom_norm

# AFTER (calc-fix) — 2단계 그린블라트 + 모멘텀 오버레이
# Stage 1: 절대 등수 합산 (단위 무관, 임의 가중 없음)
df['greenblatt_rank'] = value_rank + df['rank_roe']   # large_cap

# Stage 2: [0,1] 정규화 후 momentum_weight 비율로 혼합
df['rank_greenblatt_pct'] = df.groupby('market')['greenblatt_rank'].rank(ascending=True, pct=True)
df['total_rank'] = (1.0 - m) * df['rank_greenblatt_pct'] + m * df['rank_mom_pct']
```
